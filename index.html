Природные данные

1

💬Коммуникативные способности

Что дано от рождения:

  • Лёгкость в установлении контакта
  • Чтение социальных сигналов
  • Поддержание беседы
  • Активное слушание
2

🎭Эмоциональный интеллект

Природные преимущества:

  • Выразительная мимика
  • Естественная жестикуляция
  • Эмоциональная заразительность
  • Чувствование аудитории
3

🦁Уверенность

Врождённые особенности:

  • Низкий страх оценки
  • Комфорт в центре внимания
  • Стрессоустойчивость
  • Быстрое восстановление
4

🎤Голосовые данные

Природные характеристики:

  • Приятный тембр
  • Хорошая дикция
  • Широкий диапазон
  • Выносливость связок
5

✨Креативность

Естественные способности:

  • Импровизация
  • Генерация идей
  • Остроумие
  • Гибкость мышления

Программа развития

1

🔗Развитие коммуникабельности

Упражнения для группы 1:

  • Правило 3-х контактов — ежедневные беседы с незнакомцами
  • Активное слушание — минимум 3 уточняющих вопроса
  • Игра "Интервью" — беседа на незнакомую тему
2

👐Тренировка эмоциональности

Упражнения для группы 2:

  • "Зеркало эмоций" — одна фраза, 5 настроений
  • Анализ жестов — запись и разбор видео
  • "Эмоциональный словарь" — описание предметов с разной окраской
3

🛡️Повышение уверенности

Упражнения для группы 3:

  • "Минута смелости" — выход из зоны комфорта
  • Метод "5 почему" — анализ страхов
  • Игра "Провал" — преднамеренные ошибки
4

🎙️Постановка голоса

Упражнения для группы 4:

  • "Дыхание спикера" — 5 минут диафрагмой
  • Скороговорки-марафон — 3 скороговорки × 5 раз
  • "Радиоведущий" — 10 минут непрерывной речи
5

🎨Развитие креативности

Упражнения для группы 5:

  • Метод "Да, и..." — развитие чужих идей
  • "Безумные связи" — соединение случайных предметов
  • Сторителлинг — мини-истории по случайному слову
